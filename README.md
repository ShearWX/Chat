Basic Peer to Peer message website to use in school or anywhere.

CREATING A ROOM:
Create a room by typing a name and clicking "Create Room"
You'll receive a code to share with people so they can join

JOINING A ROOM:
You type a name and then the room owner will show/send you an invite code to input in the "Invite Code" area (pretty complicated am i right?)
Once the code is inputed you'll recive an anwser code to show to the owner who will input that anwser code to let you join
